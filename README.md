# Usage
<!-- usage -->
```sh-session
$ npm install -g yaml-config-lint
$ ycl --help
Lint multiline data inside YAML files

USAGE
  $ ycl

OPTIONS
  -f, --files=files  (required) Input File(s)
  -h, --help         show CLI help
  -k, --keys=keys    (required) Keys you want to validate
  -v, --version      show CLI version
...
```
